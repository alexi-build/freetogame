# FreeToGame

Discover free-to-play games from FreeToGame directly in Raycast.

## Features

- Search free-to-play games by title, genre, platform, publisher, or developer.
- Filter games by platform, category, release date, or alphabetical order.
- Preview game details with thumbnails, descriptions, genre, platform, publisher, developer, and release date.
- Open a game's website or FreeToGame profile from Raycast.
- Retry loading if the game list cannot be fetched.

Data is provided by [FreeToGame.com](https://www.freetogame.com).
