# FreeToGame

Discover free-to-play games from FreeToGame directly in Raycast.

## Features

- Search free-to-play games by title, genre, platform, publisher, or developer.
- Filter games by platform or category.
- Sort games by release date, popularity, alphabetical order, or relevance.
- Preview full game details with thumbnails, descriptions, screenshots, genre, platform, publisher, developer, release date, status, and system requirements.
- Toggle detail metadata with Quick Look and open a game's website or FreeToGame profile from Raycast.
- Cache game details by ID and retry loading if the game list cannot be fetched.

Data is provided by [FreeToGame.com](https://www.freetogame.com).
